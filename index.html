nana Scraper

What's nana?

nana is a music collab app where you can record your singing and play by smartphone. You can communicate with users by "collab" which you can overlap sounds, or commenting users posted sounds.

How does this work?

Every request happens on client-side using *CORS Proxy API then processed with RegEx expressions to retrieve data from the response.
